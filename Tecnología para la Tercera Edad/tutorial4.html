Instrucciones para agregar contactos en un teléfono móvil

Descripción del Video:

En esta sección, el presentador explica los pasos para agregar nuevos contactos a un teléfono móvil.

Específicamente cómo agregar un contacto para realizar una llamada telefónica o usar WhatsApp.

Pasos para agregar un contacto:

  1. Busca el ícono de un teléfono verde, en el cual debe aparecer un teclado numérico.
  2. Luego, ingresa el número de teléfono deseado en el teclado.
  3. Para crear un nuevo contacto, toca el ícono y luego escribe el nombre del contacto en el campo "Nombre".
  4. Una vez guardado, el contacto se puede utilizar para realizar una llamada telefónica o utilizar WhatsApp.

Pasos para enviar un mensaje en WhatsApp:

  1. Abre la aplicación WhatsApp.
  2. Toca el ícono de una lupa.
  3. Luego, ingresa el nombre del contacto en el campo "Para".
  4. El mensaje se puede enviar haciendo clic en el botón "Enviar", que luego puede enviar fotos, notas, notas de voz o videos.
