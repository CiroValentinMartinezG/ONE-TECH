Habla con boti

Descripción del Video:

Pasos para comunicarse con Bot:

  1. Si el usuario no está seguro de cómo agregar el contacto, puede consultar un tutorial que explica el proceso.
  2. Para comunicarse con Bot, el usuario debe abrir la aplicación WhatsApp y buscar a Bot usando el número o nombre.
  3. Una vez abierto el chat con Bot, el usuario debe tener en cuenta los siguientes tres puntos:
    • Utilizar frases cortas en lugar de explicaciones extensas.
    • No escribir palabras innecesarias en el chat.
    • Desplazarse hacia arriba y hacia abajo por el chat para leer la respuesta completa.
  4. Luego, el usuario puede escribir su mensaje y presionar el botón "shift" para comunicarse automáticamente con Bot y ver una lista de opciones en el chat, incluidas propuestas especiales e información sobre diferentes procedimientos.
  5. El usuario puede elegir un tema escribiendo la letra correspondiente a su opción deseada, y Bot le proporcionará la información y ayuda adecuadas para ese tema.
  6. Si el usuario no encuentra la información que necesita, siempre puede optar por chatear con un operador escribiendo la letra "si" y regresando al menú principal para encontrar la opción "chatear con un operador".
